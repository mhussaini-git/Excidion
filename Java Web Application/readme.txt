For this Project to Work, You will be needing the following things
1. Eclipse Photon 2018 or Above
2. Java 8 Runtime Enviroment
3. SQLITE3 v3.31
4. Tomcat 8 or Tomcat 8.5
5. Sqlite JDBC Driver 3.30.1

Steps:
1. Open Eclipse
2. Go to File -> Open Existing Project
3. Select Directory of the Project
4. Select Web and War file
5. Finish
